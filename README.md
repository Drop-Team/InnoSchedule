## Running

Requirements:
Docker

Clone repository:
```shell
git clone https://github.com/InnoSWP/InnoSchedule.git
```

Setup environment:
1. Rename `.env-example` to `.env`.
2. Edit it.

Run Docker containers:
```shell
docker-compose build
docker-compose -d up  # -d to run in background
docker-compose ps
```
